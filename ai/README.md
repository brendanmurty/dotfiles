# AI Code Generation Tools

## Install Script

To install and update popular AI tools, run [ai-install.sh](ai-install.sh):

```bash
bash ./ai-install.sh
```

## Bookmarks

- [Vercel's Skills Tool](https://github.com/vercel-labs/skills) - Simple tool to install other AI skills at a project or system level.
- [Matt Pocock's Skills](https://github.com/mattpocock/skills) - AI skills that are designed for `Real Engineers`.
